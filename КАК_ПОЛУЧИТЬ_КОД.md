# Как получить код автоматически

## Что у вас есть:
✅ Session файл: `573025288905.session`  
✅ JSON файл: `573025288905.json`  
✅ API credentials в JSON: `app_id=2040`, `app_hash=b18441a1ff607e10a989891a5462e627`

## Что нужно сделать:

### 1. Создать .env файл (в корне проекта)

Создайте файл `.env` в папке `C:\Users\User\Desktop\project\bip\`:

```
TELEGRAM_API_ID=2040
TELEGRAM_API_HASH=b18441a1ff607e10a989891a5462e627
```

**ИЛИ** система автоматически возьмет их из вашего JSON файла.

### 2. Как получить код для НОВОГО номера:

1. Откройте http://localhost:8001/sessions
2. Нажмите "Получить код"
3. Введите **НОВЫЙ** номер телефона (НЕ 573025288905, а другой!)
4. Нажмите "Запросить новый код"
5. Система автоматически:
   - Использует вашу сессию (573025288905) для мониторинга Telegram
   - Парсит ВСЕ чаты и диалоги
   - Находит код для нового номера
   - Автоматически заполняет код на сайте

### 3. Важно:

- ❌ Если вы запрашиваете код для **573025288905** - система скажет "уже авторизован"
- ✅ Запрашивайте код для **ДРУГОГО** номера (например, +79001234567)
- ✅ Система использует вашу сессию (573025288905) для мониторинга
- ✅ Код автоматически появится на сайте

## Проверка:

1. Сервер запущен на http://localhost:8001
2. Откройте http://localhost:8001/sessions
3. Попробуйте запросить код для нового номера
4. Код должен автоматически появиться!
